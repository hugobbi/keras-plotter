# mnist-visual-fields

Code to handle creation and visualization of neural networks using Keras. It was made to work using the MNIST dataset, and the neural networks can have one or two visual fields.

## Dependencies
* tensorflow v2.15.0.post1
* keras v2.15.0
* matplotlib v3.8.3
* numpy v1.26.4
* attrs v23.2.0
